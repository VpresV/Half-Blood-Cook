Οδηγίες Χρήσης

Πώς να προσθέτεις νέες συνταγές στο Half-Blood Cook.

🔁 Η Ροή (κάθε φορά)

Βήμα 1: Άνοιξε το admin.html τοπικά στον browser σου (double-click ή drag στο Chrome).

Βήμα 2: Συμπλήρωσε τη φόρμα "Νέα Συνταγή" με τα στοιχεία.

Βήμα 3: Κάνε κλικ "Δημιούργησε Αρχείο Συνταγής" → κατεβάζει recipe-τιτλος.html.

Βήμα 4: Κάνε κλικ "Δημιούργησε JSON Entry" → αντέγραψε το block.

Βήμα 5: Άνοιξε το recipes.json και πρόσθεσε το νέο entry στη λίστα (πριν το ]). Πρόσθεσε κόμμα μετά το προηγούμενο entry.

Βήμα 6: Ανέβασε στο GitHub το νέο .html αρχείο μέσα στο φάκελο recipes/ ΚΑΙ το ενημερωμένο recipes.json. Done!

📁 Δομή Αρχείων

half-blood-cook/

    ├── mosxari-stifado.html
    ├── tzatziki.html
    └── ... νέες συνταγές εδώ

⚠️ Σημαντικό

  • Το admin.html είναι μόνο για τοπική χρήση — μην το ανεβάζεις στο GitHub.
  • Κάθε νέα συνταγή χρειάζεται: ένα .html αρχείο στο recipes/ + ένα entry στο recipes.json.
  • Το index.html φορτώνει αυτόματα τις συνταγές από το recipes.json — δεν χρειάζεται να αγγίξεις το index.html.
  • Featured συνταγή: μόνο μία μπορεί να είναι "featured": true — αλλιώς εμφανίζονται όλες.

🌐 GitHub Pages Setup

  1. Δημιούργησε repo: github.com/new → όνομα: half-blood-cook
  2. Ανέβασε όλα τα αρχεία (εκτός admin.html)
  3. Settings → Pages → Branch: main → / (root) → Save
  4. Σε ~2 λεπτά: https://[username].github.io/half-blood-cook/

🔗 Custom Domain

  1. Αγόρασε domain (Namecheap, GoDaddy, Papaki κτλ.)
  2. GitHub repo → Settings → Pages → Custom domain → βάλε το domain
  3. Στο DNS του registrar: 4 A records → 185.199.108.153, .109.153, .110.153, .111.153
  4. CNAME record: www → [username].github.io
  5. Ενεργοποίησε "Enforce HTTPS"
